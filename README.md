# PyPack
My Personal Python Package

---

## Fonctions Included:

### UI Related
- **Error Message:** Prints a given str in red
- **Success Message:** Prints a given str in Green
- **Highlighted Message:** Prints a given str in Yellow
- **Wait:** Pause the program for the given int in seconds

### File & Path
- **Existing Directory:** Returns true if the dir exist, has an option to create it if it doesnt
- **Existing File:** Returns true if a file exist

### Collections (List, Tuples, Sets, Dict)
- **Flatten Nested List:** Flattens nested lists/tuples into a single list

### Strings 
- **Is Blank:** Returns true for None, empty or whitespace only string

### Numbers (int, float, hex, ...)
- **Convert Number:** Converts given number into desired base
